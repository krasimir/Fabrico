Configuration of Fabrico

The configuration of Fabrico is stored in /config/config.json. It has the following structure: 

{
    "access": [access_object],
    "paths": [path_object],
    "routes": [
        [route_object], 
        [route_object], 
        ...
    ],
    "adapters": {
        "[adapter_name]": [adapter_object]
    }
}

Access object

{
    "user": "admin",
    "pass": "admin"
}

Path object

{
    "files": "/Fabrico",
    "http": "/fabrico"
}

Route object

{
    "url": "[regular_expression]", 
    "controller": "[path_to_controller_php]", 
    "model": "[path_to_model_json]" // optional
}

MySQL adapter object

"MySQL": {
    "host": "localhost",
    "user": "root",
    "pass": "",
    "dbname": "fabrico_test"
}


Example:

{
    "access": {
        "user": "admin",
        "pass": "admin"
    },
    "paths": {
        "files": "/Fabrico",
        "http": "/fabrico"
    },
    "routes": [
        {"url": "/sample(.*)?", "controller": "pages/Page.php", "model": "models/sample.json"},
        {"url": "/logout(.*)?", "controller": "pages/Logout.php"},
        {"url": "/login(.*)?", "controller": "pages/Login.php"},
        {"url": "(.*)?", "controller": "pages/Home.php"}
    ],
    "adapters": {
        "MySQL": {
            "host": "localhost",
            "user": "root",
            "pass": "",
            "dbname": "fabrico_test"
        }
    }
}
