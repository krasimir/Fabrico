Usage:
  • routes [show] [unit] [config file] - show models
    • [show] (mandatory)
    • [unit] (optional) - name of a unit. Default value: admin
    • [config file] (optional) - path to the configuration file. Default value: modules/router.php
  • routes [manage] [unit] [config file] - add/edit/delete routes
    • [manage] (mandatory)
    • [unit] (optional) - name of a unit. Default value: admin
    • [config file] (optional) - path to the configuration file. Default value: modules/router.php
