Usage:
  • Controllers [show] [unit] [config file] - show models
    • [show] (mandatory)
    • [unit] (optional) - name of a unit. Default value: admin
    • [config file] (optional) - path to the configuration file. Default value: modules/controllers.php
  • Controllers [manage] [unit] [config file] - add/edit/delete controllers settings
    • [manage] (mandatory)
    • [unit] (optional) - name of a unit. Default value: admin
    • [config file] (optional) - path to the configuration file. Default value: modules/controllers.php
